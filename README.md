# i-SSH

一个提供 ssh 命令行输入的 web 应用程序.

## 功能特点

* web ssh
* 服务器信息管理
* 多机器同步输入
* 免跳板机

## 账户注册

> 用户初次使用可选择 `个人账户` 或 `域账户` 登录，若没有 `个人账户`，需在主页面点击 `Register` 按钮进行账户注册。   

![Register Page](/raw/master/src/docs/images/register.png)

- `UserName`: 新注册账户名称      
- `Password`: 账户密码   
- `Display Name`: 登录后显示名称   
- `Email`: 账户邮件地址
